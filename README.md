# AWS Route53 Clone

A full-stack Route53 Console clone built for the Scaler AI Labs Software Engineering Intern assignment.

## Tech Stack

- Frontend: Next.js (TypeScript)

- Backend: FastAPI

- Database: SQLite

- ORM: SQLAlchemy

- Styling: Tailwind CSS

## Features

- Mock Authentication

- Hosted Zone CRUD

- DNS Record CRUD

- Search

- Pagination

- Dashboard

- AWS Console inspired UI

## Run Backend

cd backend

venv\Scripts\activate

uvicorn main:app --reload

## Run Frontend

cd frontend

npm install

npm run dev
